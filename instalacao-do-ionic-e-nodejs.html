Instalação do Ionic e Node.js

Seg 26 outubro 2015 Por Paulo Miranda Moreira

Estou aprendendo Ionic, um excelente framework para desenvolvimento de apps móveis híbridos (para IOS e Android).

Tive alguns problemas para instalação, pois o Ionic depende do Node.js, que estava no repositório do Linux Mint com uma versão desatualizada.

Tentei baixar diretamente do repositório no Github, mas lá só havia a versão de desenvolvimento, e, mesmo conseguindo instalá-la, tive problemas para instalar o Ionic mais tarde.

Achei a solução aqui e reproduzo os passos abaixo. Estou utilizando Linux Mint 17.

No terminal, configuração e instalação do Node.js:

curl --silent --location https://deb.nodesource.com/setup_0.12 | sudo bash -
sudo apt-get install --yes nodejs

Instalação do Ionic:

sudo npm install -g ionic

Simples, não? Espero que esse post economize sua dor de cabeça!

Importante: se você já instalou o Node.js a partir do código-fonte do repositório do github, antes de instalar o Ionic execute o seguinte comando na pasta clonada do github:

sudo make uninstall
