Configurando bluetooth no Linux Mint

Qui 04 fevereiro 2016 Por Paulo Miranda Moreira

Quando o adquiri meu computador (há uns 4 anos) tentei rapidamente configurar o bluetooth, mas, por não ter o costume de utilizá-lo, acabei deixando de lado. Esses dias ganhei uma caixa de som que utiliza o bluetooth para se conectar, então novamente tentei configurá-lo, mas não tive sorte. Após instalar o blueman, ao tentar conectar com o equipamento de áudio sempre via o erro “stream setup failed”. Finalmente, achei [essa dica] (http://firstdoit.com/quick-tip-bluetooth-a2dp-on-linux-mint-17-qiana-mate/), lembrando que vale para Linux Mint 17 Qiana MATE:

Instalar os módulos recomendados: :::shell sudo apt-get install blueman bluez pulseaudio-module-bluetooth —install-suggests pactl load-module module-bluetooth-discover

Agora basta clicar no ícone do bluetooth da barra de ferramentas, selecionar Devices, e conectar o equipamento. Importante notar que, no assistente do bluetooth, ao clicar com o botão direito no equipamento conectado, em “audio profile”, é melhor selecionar a opção “High Fidelity Playback (A2DP)” para maior qualidade de som.

O inconveniente é que é necessário rodar o último comando toda vez que o computador é reiniciado. Para resolver esse problema, um comentário de Kory Wiesner no post acima sugeriu o seguinte:

MENU —> Preferences —> Startup Applications Clique em “Add”, dê um nome ao seu script e no comando coloque: bash -c “sleep 5; pactl load-module module-bluetooth-discover”.

Adicione um comentário, se quiser, feche e tenha certeza que o checkbox do comando que você acabou de criar está selecionado.

O tempo de sleep é necessário para que o sistema carregue todo o aparato necessário para que o comando seguinte seja executado. No meu caso, em vez de 5 segundos, precisei colocar 20.

Pronto! Agora desfrute do seu bluetooth!!!
